Problem Statement

Alex loves decoding secret messages, and one day he encounters an encoded string in a special format. He challenges his friend Bella, who is skilled at programming, to write a function to decode it.

Input Format

First line of input consists a integer T,denoting the number of testcases

Each test case consists of single line of input which the string that needs to be decoded

Output Format

Output the decoded string

Example

Example 1:

Input:

2

3[a]2[A3[b]]

4[K]5[T]

Output:

aaaAbbbAbbb

KKKKTTTT
